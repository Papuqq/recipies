# key_project_4.6_recipes

## run back
- open a terminal


- activate the virtual environment

    `python3 -m venv venv`

    `source venv/bin/activate`


- install packages

    `pip install -r requirements.txt`

    `cd recipes`


- run server

    `python3 manage.py runserver`

## API documentation
open: http://127.0.0.1:8000/swagger/

## django admin
open: http://127.0.0.1:8000/admin/
- login: admin
- pass: admin

## run front

- open new terminal

  `cd recipes_front`

  `npm install`

  `npm start`
